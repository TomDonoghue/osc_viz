EEGVis: A Set of Interactive Visualizations for EEG Data

Electrophysiological recordings, such as electroencephalography (EEG), magnetoencephalography (MEG) and electrocorticography(ECoG) are increasingly prevalent tools in the brain sciences, and yet they can be difficult tools to learn.

Here a set of short tutorials and interactive visualizations have been designed to try and foster increased understanding of these methods.

  • Oscillations
  • Event-Related Potentials
  • Adding Waves Together
  • Averaging ERPs
